sh404SEF plugin for OS CCK

SEO Meta Tags plugin for OS CCK - free OS CCK software for download and available in Joomla 3.5. This best SEO (search Engine optimization) and SEM (Search engine marketing) tool for OS CCK software.

This extension file for sh404SEF (sh404SEF native plugin) enables generation of Search Engine Friendly titles and meta tags(meta keywords and meta descriptions) for free software OS CCK. Automatically generated Title and meta keywords and meta descriptions for every Instance are composed from the categories and titles.


Requirements:

Joomla 3.5.x, PHP5, OS CCK 2.5.4 Pro, sh404SEF 4.8.2.3492

Installation:

    * Now using FTP or similar, drop the file com_os_cck.php into that meta_ext directory
    * or you may upload this file to /components/com_sh404sef/meta_ext folder 	
    * Now you have very nice SEF Titles and meta keywords and meta descriptions for every Instance for OS CCK in the frontend.
